    Required CSV Format

    Your CSV file should contain individual columns for customer/loan data, and single columns for raw bureau and financial data.

    "loan_id","cust_id","age","gender","loan_amount","bureau_xml","mobile_json","bank_json"
"AF7737745","KF3999955",34,"F",50000,"<crb>...</crb>","{...}","[{...}]"
...

    Note: The data inside the `mobile_json` and `bank_json` columns must be valid JSON strings. The `bureau_xml` column should contain the raw XML or text from the bureau report.
